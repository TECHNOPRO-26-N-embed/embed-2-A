# timer.ino テスト仕様書（詳細設計書ベース）

## 1. 目的
- 詳細設計書の各関数・状態・分岐が仕様通りに動作することを確認する。
- 関数単位の単体テスト観点を明確化し、網羅的にテストできるようにする。

## 2. テスト対象
- 対象ファイル: timer.ino
- 対象関数: setup, loop, readKeypad, setTimeFromKeypad, showTimeSerial, updateLedByState, notifyByBuzzer, updateNotify, displayTime, playMelody

## 3. テスト環境
- Arduino Uno 実機
- 4x4キーパッド、青/赤LED、パッシブブザー、7セグ（任意）
- シリアルモニタ 9600bps

## 4. テストケース一覧

| ID | 観点 | 手順 | 期待結果 |
|----|------|------|----------|
| S-01 | 初期化 | 電源ON/リセット | ピンモード/シリアル/LED初期化が行われる。青LEDが1秒点灯後消灯 |

### 4.2 loop() 状態遷移
| ID | 状態 | 入力 | 期待遷移・動作 |
|----|------|------|----------------|
| L-01 | 待機中 | 数字→A | 計測中へ遷移、remainSecondsに値セット |
| L-02 | 計測中 | 1秒経過 | remainSecondsが1減、0で終了通知へ |
| L-03 | 計測中 | B/C | 待機中へ遷移、remainSeconds=0 |
| L-04 | 終了通知 | B/C | 待機中へ遷移、LED/ブザーOFF |
| L-05 | 終了通知 | 無効キー | 状態継続 |
| L-06 | 不正state | 強制書換 | 待機中へ復帰 |

【記入欄】


### 4.3 readKeypad()
| ID | 観点 | 手順 | 期待結果 |
|----|------|------|----------|
| K-01 | 有効キー | 任意キー押下 | 対応文字が返る |
| K-02 | 無効キー | 存在しないキー押下 | '\0'が返る |
| K-03 | デバウンス | 連打 | チャタリング抑制で重複検出しない |

【記入欄】


### 4.4 setTimeFromKeypad()
| ID | 入力 | 期待結果 |
|----|------|----------|
| T-01 | 数字 | バッファに追加、範囲内ならOK |
| T-02 | A(有効) | 設定確定、true返却 |
| T-03 | A(無効) | 設定不可、false返却 |
| T-04 | C | バッファクリア |
| T-05 | 上限超過 | 5999超は無効 |

【記入欄】
